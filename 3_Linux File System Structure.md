Linux has only ONE main folder, called root /

Everything is inside /

/
├── bin
├── etc
├── home
├── root
├── var
├── tmp

Important Folders Explained (Very Simple)
Folder	Meaning
/	Main root
/home	Normal users data
/root	Admin user data
/etc	Configuration files
/bin	Basic commands
/var	Logs
/tmp	Temporary files
Example:
ls /


This lists all main folders
